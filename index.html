  • Cloud

    Тестовое задание:

    1. Стек: react, typescript, redux toolkit, formik, yap
    2. Разработка формы описания профиля
    В рамках дизайна формы реализовано 3 отдельных таба(шага) формы, которые можно переключать между собой. При переходе от таба к табу (в том числе возвращаясь на предыдущий) информация должна сохранятся. На первом экране необходимо добавить информацию о себе и по нажатию на кнопку будет происходить переход на форму. При переходе должен меняться роут. На каждом этапе формы нужно валидировать значения конкретного шага. После отправки формы показывать модальное окно с success или error. Модалку нужно будет разработать самому, не используя сторонние библиотеки или ui-компоненты. Для отправки формы использовать api https://api.sbercloud.ru/content/v1/bootcamp/frontend. Адаптивная версия обязательна, способ реализации по твоему выбору
